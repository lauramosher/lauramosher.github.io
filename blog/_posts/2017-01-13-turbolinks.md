[Turbolinks](https://github.com/turbolinks/turbolinks) takes advantage of a lot of benefits of a single page application without needing to actually build or add the complexity of a client-side JavaScript framework. One of the ways it achieves the snappy page loads of a single page application in a traditional, link-based application is caching. When a page is visited, Turbolinks will cache the final state of the visited page. That includes caching the final markup of the document, the stylesheets, image assets, and JavaScript.
